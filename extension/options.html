◉ PhishGuard Extension Settings

Configure connection to your PhishGuard server.

Server Connection

The URL where your PhishGuard instance is running.
Found in your PhishGuard account settings. Supports both legacy API keys and JWT tokens.

How to Use

Right-click analysis: Right-click on any email in Gmail or Outlook Web, then select "Analyze with PhishGuard" from the context menu.

Toolbar button: When viewing an email, click the "Analyze" button that appears in the email toolbar.

Popup upload: Click the PhishGuard icon in your browser toolbar to upload .eml files directly.

Supported inboxes: Gmail, Outlook Web (outlook.live.com, outlook.office.com, outlook.office365.com).
